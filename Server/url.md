# URL : Uniform Resource Locator

## Example: https://www.pragyan.dev/

Protocol: HyperText Transfer Protocol Secure
Here, https:// is Protocol. http is less secure than https

Domain: User Friendly Name of IP Address of Server
www.pragyan.dev is the domain.

Path: / : Home
/about : for about page
Nested Path:
/project/1
/Pragyan-Misra/node/File_Handelling

# Query Parameter: pragyan.dev/about?userid=1&a=2

page opening is about and we have passed some extra info that userid is 1 and a = 2.
Query Parameter starts with "?" and all spaces are converted into "+"
Example: https://www.youtube.com/results?search_query=node+js
